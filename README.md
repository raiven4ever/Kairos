# kairos

**kairos** is a roblox studio plugin for tracking the time you spend developing. it runs quietly in the background, recording work sessions and accumulating statistics so you always know where your hours are going.

## how it works

kairos starts tracking the moment studio opens. assign time to a **project** — a simple, freeform container you define — and it will record every session, calculate totals, and surface stats like your longest session, active days, and average work length.

if you haven't selected a project yet, time is held in an **unassigned** pool and can be reassigned later.

## features

- **automatic session tracking** — starts on launch, continues where you left off
- **flexible projects** — name them, describe them, tag them however makes sense to you
- **rich statistics** — total time, session count, averages, longest session, active days, and more
- **filtering & sorting** — find projects by name, tags, date ranges, time spent, and other metadata
- **persistent storage** — all data is saved locally via studio plugin settings

## philosophy

kairos is deliberately simple and non-opinionated. projects are just containers for time — what they represent is entirely up to you.
